Tipo de Proposição

Esta tabela tem por objetivo cadastrar os Tipos de Proposições Legislativas.
Ao ter acesso a esta tabela será exibida uma tela com a relação de Tipo de Proposições já cadastradas e um botão “Incluir Tipo de Proposição“.

Ao ser  acionado o botão Incluir Tipo Proposição será exibida a tela com os seguintes campos a serem preenchidos:

Nome do Campo

Descrição

Instruções para Preenchimento

Descrição

Obrigatório

Faça breve descrição sobre o tipo de proposição em questão.

Indicação - Gera Matéria ou Documento

Obrigatório

Assinale se o tipo que está sendo criado poderá gerar uma matéria ou um Documento.

Tipo Matéria ou Documento

Obrigatório

Selecione conforme foi anteriormente assinalado.

Modelo XML

Obrigatório

Selecione o modelo XML que estará associado ao Tipo que está sendo criado.


Acione a função  “Salvar” para que as informações sejam salvas no arquivo.
Será enviada a mensagem !Tipo Proposição salvo com sucesso!

Acione a função Continuar para  repetir a operação para outros Tipo de Proposição, ou acione a função “Início“ para retornar a lista de Tipo de Proposições.

Para alterar um Tipo Proposição, selecione clicando sobre o Tipo de Proposição desejado, altere as informações e acione a função “Salvar”.

Para excluir um Tipo de Proposição, selecione clicando sobre o Tipo e acione a função “Excluir”. Será enviada a pergunta “Deseja realmente excluir?”
Acione OK em caso positivo ou Cancelar para o caso de desistência da exclusão.
Quando for informado OK será enviada a mensagem !Tipo Proposição excluído com sucesso!
A função “Início” retorna a tela com a relação de tipo de proposições já cadastradas.
